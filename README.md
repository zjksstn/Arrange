# 瑞士制棋类比赛编排系统

一套面向中国象棋等棋类赛事的**瑞士制积分编排**桌面软件:从选手报名、抽签编排、现场登分,到计算最终名次、打印各类报表,一条龙完成。界面参考"云蛇比赛编排",采用三栏式布局。

> 现行版本为 **PySide6 (Qt)** 重写版(`app_qt.py`)。仓库内另保留了早期 tkinter 版(`dataclass.py`)作为兜底,不再维护。

---

## 主要功能

- **赛事 / 组别两层结构**：一个赛事可含多个独立组别(如公开组、女子组、少儿组),各自名单、轮次、进度互不影响,左栏树切换。
- **报名阶段**：花名册表格直接录入,或从 **txt / Excel(.xlsx)** 导入(Excel 带列映射向导);字段含姓名、性别、团队、身份证、电话、等级分、K值。
- **抽签开赛**：可选附加条件(仅作用首轮)——*同团队首轮不相遇*、*签号小者持红先走*;后续轮完全依算法。
- **配对算法**(`pairing.py`,纯函数可单测):积分就近配对 + 回溯避免重复对阵 + 奇数自动轮空(同人不重复轮空)。
- **可视化登分**：每台一张卡片,红胜 / 和 / 黑胜一键登分。
- **全程可逆**：首轮可「重新抽签」,中间轮可「撤销本轮」,末轮「计算最终成绩」后可「撤销结算」;历史轮自动锁定。
- **退赛 / 复赛**：开赛后可让选手中途退赛或复赛,当前轮自动重排。
- **破同分**：积分 → 对手分 → 后手局数 → 犯规数 → **累进分** → 签号;成绩表对靠后手/犯规/累进分分出的名次**标红高亮**并注明依据。
- **报表(A4 PDF,跨平台、内置宋体免装字体)**：本轮对阵表、最终成绩表、选手花名册、台号卡、记分卡、荣誉证书。生成即存档到 `events/`,并调系统默认 PDF 程序打开打印。

## 运行

已配好 `venv` 的机器,直接双击:

- **`run_qt.bat`** —— 正式版。
- **`run_qt_dev.bat`** —— 开发版(多一个「随机结果(测试)」按钮,用于快速灌数据测试)。

## 从零搭建(新机器 / clone 之后)

仓库不含 `venv`(体积大、与机器相关),需自行重建:

```bat
python -m venv venv
venv\Scripts\python -m pip install --upgrade pip
venv\Scripts\python -m pip install pyside6 reportlab pillow openpyxl pypdf
venv\Scripts\python app_qt.py
```

- Python 3.10+。
- 跨平台:Windows / macOS 均可(打印调用系统默认 PDF 程序;现代 Win 用 Edge/Chrome、Mac 用预览,无需额外软件)。
- 旧 tkinter 版若要跑,另需 `pip install pinyin wxpython`。

## 目录结构

| 路径 | 说明 |
|---|---|
| `app_qt.py` | Qt 主程序(界面 + 交互) |
| `tournament.py` | 数据模型(Event/Group/Player)+ JSON 存档 |
| `pairing.py` | 瑞士制配对算法(纯函数,`python pairing.py` 跑自测) |
| `print_pdf.py` | 各类报表 PDF 生成(reportlab) |
| `sim_backtrack.py` | 配对回溯需求的蒙特卡洛分析 / 回归测试 |
| `荣誉证书底图.png` | 荣誉证书底图(可替换为更高清同名文件) |
| `台号卡底图.png` | 台号卡底图(可选;不放则为素卡) |
| `events/` | 运行时赛事存档(JSON,**不纳入版本库**) |
| `mathdate/` | 旧版历史数据(首次启动用其名单做示例赛事) |
| `dataclass.py` 等 | 早期 tkinter 版,保留兜底 |

## 数据与备份

- 每个赛事存为 `events/<赛事名>.json`(含全部组别、选手、各轮对阵与成绩)。
- `events/` 未入库;如需用 git 备份赛事数据,删除 `.gitignore` 中的 `events/` 一行即可。
